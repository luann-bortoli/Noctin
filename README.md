Noctin é uma aplicação fullstack que permite criar, editar e excluir tarefas.
O projeto é construído com React + TypeScript no frontend e Python Flask + SQLAlchemy no backend.

# Funcionalidades
- Criar novas tarefas com título, conteúdo e prioridade
- Editar tarefas existentes
- Excluir tarefas
- Ordenação por prioridade e data de criação
- Visualizar histórico de interação

# Tecnologias usadas
- React
- TypeScript
- Axios
- Python
- Flask
- SQLAlchemy
